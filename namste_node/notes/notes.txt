5 Steps to run the code #require(/path)
  1. Resolving the module
      -> Checks whether it is a local/json/node internal module path
  2. Loading the module
        -> File content is loaded according to file type
  3. Wraps inside IIFE
      -> compile step
  4. Code Evaluation
      -> require , module.exports returned
  5. Caching
      -> then the module is cached
