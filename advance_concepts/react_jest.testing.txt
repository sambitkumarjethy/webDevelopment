- Installed jest
-  Installed babel dependency

- Jest  -npx jest -init

 - - (double underscore is called dounder)

 Example of test case of sum.js

 import {sum} from "../sum"
 test("Sum function should calculate the sum of two numbers",()=>{
    const result = sum(3,4);
    
    // Assertion
    expect(result).toBe(7);
 })
